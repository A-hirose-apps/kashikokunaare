# かしこくなーれ（Kashikokunaare）

🔗 公開ページ  
https://a-hirose-apps.github.io/kashikokunaare/

子どもが楽しく「ひらがな」に触れられる、やさしい学習アプリです。  
タッチすると音声で読み上げたり、ことばをまとめて聞けたり、  
小さなお子さまでも直感的に遊べるようにデザインしています。

このアプリは **個人が非営利目的で制作したもの** で、  
**どなたでも自由にご利用いただけます。**  
広告や課金要素は一切ありません。

本アプリは個人の趣味として制作しており、今後も気ままに改良や機能追加を行っていく予定です。
そのため、仕様や画面表示が予告なく変わることがありますが、あたたかく見守っていただければ幸いです。

---

## 📱 使い方

1. ブラウザで以下の URL を開くだけで使えます  
   https://a-hirose-apps.github.io/kashikokunaare/
2. ひらがなをタッチすると音声が再生されます
3. 「次へ！」ボタンを**押す**と異なる学習画面に遷移します
4. 「次へ！」ボタンを**長押し**で好きな学習画面に遷移します

## 🌱 このアプリを作った理由

うちの子は、まだ発語がゆっくりで、  
ことばに触れるきっかけを少しでも増やせたらと思い、  
このアプリを作り始めました。

ひらがなをタッチすると音が出たり、  
好きなもの（どうぶつ・のりもの・すうじ）を選んで遊べたりすることで、  
少しずつ「声を出すきっかけ」や「まねしたくなる瞬間」が増えていきました。

同じように、  
**「遊びながらことばに触れるきっかけがほしい」**  
**「無理なく、自然に発語のヒントになるものがほしい」**  
と思っているご家庭にも、  
もし役に立つことがあれば嬉しいです。

このアプリは無料で、広告もありません。  
安心して遊んでもらえるように作っています。
